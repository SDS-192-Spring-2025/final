# Function to plot storm density using filled contours
plot_storm_density <- function(df, event_type) {
  df %>%
    filter(type == event_type, !is.na(location_lat), !is.na(location_lng)) %>%
    ggplot(aes(x = location_lng, y = location_lat)) +
    borders("state") +
    geom_density_2d_filled(contour_var = "ndensity", alpha = 0.7) +
    scale_fill_viridis_d(option = "inferno", name = paste(event_type, "Density")) +
    coord_fixed(1.3) +
    theme_minimal() +
    labs(
      title = paste("Density Map of", event_type, "Events (2016–2022)"),
      caption = "Source: WeatherEvents_Jan2016-Dec2022.csv",
      x = "Longitude", y = "Latitude"
    )
}

# Call the function to generate storm density map
plot_storm_density(weather_data, "Storm")

Blog Post

Our Research Questions

  1. What are the states of dense storm activity in the United States between 2016 and 2022?
  2. Which U.S. states experienced the highest number of storm events between 2016 and 2022?

Context of the Dataset

We used a dataset with over 8.6 million weather events recorded by 2,071 airport-based weather stations across the United States between 2016 and 2022. The dataset includes different types of weather events (like rain, snow, and storms), their severity, locations, and time.

The data focused only on storm events, which were defined as weather with wind speeds of at least 60 km/h.

Findings from Our Analysis

Storm Activity Density

We wrote a custom function in R using ggplot2 and geom_density_2d_filled() to create a map showing where storms were most common. The map uses colors to show storm frequency:

  • Brighter colors (like yellow and orange) = more storms (up to 1.0)
  • Darker colors (like purple and black) = fewer storms (down to 0.0–0.1)

From the map, we found two main areas with high storm activity:

  • The Midwestern states, especially Kansas and Nebraska
  • The Northeastern U.S., around New Hampshire

In contrast, the Western U.S. (like California and Nevada) had fewer storms. According to weather research online, this makes sense — the Midwest is known as “Tornado Alley,” where different air masses often meet and cause strong storms.

Storm Activity Density (Repeated for Emphasis)

We created a custom plotting function in R to map the geographic density of storm events using geom_density_2d_filled() from ggplot2. The resulting map shows where storm events were concentrated:

  • Brighter colors (yellow, orange): indicate higher storm densities (closer to 1.0)
  • Darker colors (purple, black): indicate fewer storms (closer to 0.0)

From this map, we observed two major regions with high storm density:

  • Midwest, particularly Kansas and Nebraska
  • Northeast, particularly around New Hampshire

In contrast, Western states such as California and Nevada showed fewer storm events.

This pattern aligns with meteorological expectations, as the Midwest lies in “Tornado Alley” — a region prone to frequent, severe storm activity due to converging air masses.

Key Takeaways and Ethical Considerations

Malika:

  • Geographic Bias: Areas with more weather stations might show more events, even if they don’t actually have more storms.
  • Data Limitations: The dataset doesn’t include U.S. territories or very small/local storms, so some areas might be underrepresented.
  • Reusable Function: We made a function that can also be used to make similar maps for other weather types (like snow or rain), making the analysis easier to repeat.
  • Privacy: This dataset doesn’t include personal data, but it’s important to be careful when working with location data that might connect to sensitive places.

Eva:

  • Geographic Bias: States with more weather stations might show more storm events simply because there are more sensors, not necessarily because they experience more storms.
  • Data Limitations: The dataset excludes U.S. territories and may not capture hyperlocal weather events.
  • Reusable Code: Our function is reusable for visualizing other weather types (e.g., rain, snow), enabling efficient future analyses.
  • Privacy Concerns: While this dataset doesn’t contain personal identifiers, any project involving geolocation data should be aware of privacy risks, especially if locations can be linked to sensitive infrastructure or communities.
